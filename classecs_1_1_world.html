◆ addStorage()

template<std::derived_from< Component > C>
getStorageType<C>& ecs::World::addStorage ( )
inline

Registers a component's storage.

Normally, component storages are auto-registered when the EntityBuilder::with() method is called. But you wouldn't be able to use ecs::maybe() with a Component that is not used by any entity in the world.

Use this function when you want to perform optional joins with the component C that might not be present in the current entities.

Template Parameters
CThe component type to register.
Returns
A reference to the Component's storage.
