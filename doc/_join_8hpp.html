Detailed Description

Provides the join() function that allows one to iterate over multiple components and resources at once.

The join() function currenly accepts:

  • Entities: fetched from SystemData::getResource<Entities>().
  • MapStorage: fetched from SystemData::getStorage().
  • MarkerStorage: fetched from SystemData::getStorage().
  • MaybeJoin: created by maybe().
