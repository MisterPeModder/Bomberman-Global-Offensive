◆ maybe()

template<Joinable J>
MaybeJoin<J> ecs::maybe ( J &  joinable)

Creates A Joinable structure that yields all indices, returning nullptr for all missing elements and pointers to the elements otherwise.

Note
Do not have a join of only MaybeJoins. Otherwise the join will iterate over every single index of the bitset. If you want a join with all MaybeJoins, add an Entities resource to the join as well to bound the join to all entities that are alive.
Template Parameters
